  1. 合并两个有序数组

   + 语言：java

   + 思路：判断nums1和nums2的数字大小，然后插入；如果还有剩余，再统统放到nums1的尾部

   + 代码(1ms)：

     ```java
     class Solution {
         public void merge(int[] nums1, int m, int[] nums2, int n) {
             int i,j;
             for(j = 0,i = 0; j < n && i < m+n; ++i){
                 if(nums1[i]>nums2[j]){
                     insert(nums1,nums2[j],i);
                     ++j;
                 }
             }
             while(j<n){
                 nums1[m+j] = nums2[j];
                 ++j;
             }
         }
         /* 插入新的数值 */
         public void insert(int[] nums,int num,int pos){
             for(int i = nums.length-1; i > pos; --i){
                 nums[i] = nums[i-1];
             }
             nums[pos] = num;
         }
     }
     ```

   + 参考代码(0ms)：巧妙利用双尾指针，不用插入，直接重新赋值数组的每个位置。

     ```java
     class Solution {
         public void merge(int[] nums1, int m, int[] nums2, int n) {
             int c = n+m-1;
             n--;m--;
             while(m>=0 && n>=0){         
                 nums1[c--] = nums2[n]>nums1[m] ? nums2[n--] : nums1[m--];
             }
             while(n>=0){
                 nums1[c--] = nums2[n--];
             }
         }
     }
     ```

   + 参考后重写：

     ```java
     class Solution {
         public void merge(int[] nums1, int m, int[] nums2, int n) {
             int cur = m+n-1;
             --m;
             --n;
             while(m>=0&&n>=0){
                 nums1[cur--] = nums2[n] > nums1[m] ? nums2[n--] : nums1[m--];
             }
             while(n>=0){
                 nums1[cur--] = nums2[n--];
             }
         }
     }
     ```

  2. 第一个错误的版本

   + 语言：java

   + 思路：二分查找，每次取中间和中间的右边那个进行判断。

   + 代码(19ms,击败23.74%，丢人，哈哈)：

     ```java
     /* The isBadVersion API is defined in the parent class VersionControl.
           boolean isBadVersion(int version); */
     
     public class Solution extends VersionControl {
         public int firstBadVersion(int n) {
             if(n>1){
                 int start = 1;
                 int end = n; 
                 int middle;
                 boolean left = false;
                 boolean right = false;
                 while(start<=end){
                     middle = start + (end-start)/2;
                     left = isBadVersion(middle);
                     if(!left){
                         right = isBadVersion(middle+1);
                         if(right)
                             return middle+1;
                         else{
                             start = middle+1;
                         }
                     }
                     else{
                         end = middle - 1;
                     }
                 }
             }
             return 1;
         }
     }
     ```

   + 参考后重写(12ms)：主要是要清楚最后返回的是end，因为end就是符合条件的那个

     ```java
     /* The isBadVersion API is defined in the parent class VersionControl.
           boolean isBadVersion(int version); */
     
     public class Solution extends VersionControl {
         public int firstBadVersion(int n) {
             int start = 1;
             int end = n;
             int middle;
             while(start<end){
                 middle = (start+end)>>>1;
                 if(isBadVersion(middle)){
                     end = middle;
                 }else{
                     start = middle + 1;
                 }
             }
             return end;
         }
     }
     ```
